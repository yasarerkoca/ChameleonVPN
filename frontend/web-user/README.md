# Chameleon Web User

## Kurulum
npm ci

## Geliştirme
Vite geliştirme sunucusunu başlatmak için:

```
npm run dev
```

## Test
npm test -- --watchAll=false

## Build
Üretim derlemesi oluşturmak için:

```
npm run build
```

Oluşturulan derlemeyi yerel olarak görüntülemek için:

```
npm run start
```
